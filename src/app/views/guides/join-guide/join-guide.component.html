Joining Guide

1 Locate a Session to Join

Either click on a link of a session posted in a thread or look for open sessions in the Session List to join. In the session list, you can search for specific session of players or filter by turnip prices.

2 Request the Dodo Code

Sessions can also be set to only allow verified players. If that is the case then a key will be displayed in the ID of the session.

To join a verified-only session, you have to get verified. Contact a verifier or moderator for details.

3a Obtain the Dodo Code

After getting accepted you may display the Dodo Code of the session.

3b Rejected

In the event that you are rejected, you will not be able to request the Dodo Code of the session again unless the host withdraws the rejection.

If you find that you are often rejected, perhaps consider contacting a verifier or moderator to verify you so that hosts can be assured you are an actual player wanting to join the activities.
