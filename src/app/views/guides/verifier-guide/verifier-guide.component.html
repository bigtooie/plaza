Verifier Guide

As a verifier, your job is to verify people. You also get a special icon on your ID, like this:

You can only set verifications of users which have not been verified yet, or users which you verified, so if another Verifier, Moderator or Admin verifies a user, you cannot remove or change that verification.

How to Verify

1 Go to the Profile of the Player

Search for the player on the Players page or simply click on the players ID when you come across it to go to the profile of the player.

2 Open the Edit Verification Dialog

3 Set the Verification

To set the verification, you can either set it to 0 (the number zero) to set a verification by trust, i.e. there is no (public) proof to show others that this player is indeed an actual ACNH player, or set it to a post number of a post containing enough proof that the player owns the ID.

When setting verification to a post number, make sure the post contains enough information to prove that the player does own the ID, e.g. they attached a picture of their character saying their ID. Make sure attached images aren't edited, e.g. the names of a player or the ID are tampered with. When in doubt, request to meet them in-game, ask them about it and verify them by trust (0).
